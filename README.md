Needed Stuff:
----------------------------------------------------------

Windows Side Loaded Linux BASH Shell

Decrypted and Unzipped Zapp Logs (preferably on desktop)


Windows:
----------------------------------------------------------

Open command prompt

Change directory to where you extracted your files to. 

To run type:

wsl ./ZappLogAnalyzer

Linux/Mac:
----------------------------------------------------------

Open terminal

Change directory to where you extracted your files to.

To run type: 

./ZappLogAnalyzer
